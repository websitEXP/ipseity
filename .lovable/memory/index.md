# Project Memory

## Core
Poetry portfolio for Ons Ferjani. Site name: **ipseity**. Tagline: "somehow everything is poetic". 4 seasons theme (default: Autumn). No bright whites—cream/off-white only.
Fonts: Cormorant Garamond headings, EB Garamond body, Dancing Script signatures.
Colors: Forest #0A3323, Moss #839958, Cream #F7F4D5, Rosy #D3968C, Midnight #105666.
Grain texture overlay on everything. Parchment texture on content sections. Arch-shaped poem cards.

## Memories
- [Season system](mem://features/seasons) — 4 seasons with unique backgrounds, colors, particles, cat mascot variants
- [Color palette](mem://design/colors) — Per-season color schemes: Autumn (forest/cream/moss), Spring (cream/rosy/moss), Summer (golden/terracotta/warm cream), Winter (midnight/ice/frost)
- [Site structure](mem://features/structure) — 4 pages: Home, Poems, About, Contact. Sleeping cat mascot bottom-right.
